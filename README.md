# 🏨 Hotel Booking Cancellation Analysis

## 📌 Project Overview

Hotel booking cancellations have become a major challenge for the hospitality industry, leading to revenue loss, inefficient room utilization, and difficulties in demand forecasting. Understanding the factors behind booking cancellations can help hotels improve occupancy rates and optimize business performance.

This project analyzes hotel booking data from **City Hotels** and **Resort Hotels** using Python to uncover booking trends, customer behavior, and the key drivers of cancellations. Through exploratory data analysis (EDA), the project provides actionable business insights and recommendations to help reduce cancellation rates and improve revenue generation.

---

## 🎯 Business Problem

Hotels experience a significant number of booking cancellations, resulting in lost revenue and underutilized rooms. High cancellation rates also make inventory planning, staffing, and pricing strategies more challenging.

The objective of this project is to analyze historical hotel booking data to identify cancellation patterns, understand customer booking behavior, and provide data-driven recommendations that can help hotel management reduce cancellations, improve occupancy, and maximize revenue.

---

## 🎯 Project Objectives

- Analyze hotel booking cancellation patterns.
- Compare booking behavior between City Hotels and Resort Hotels.
- Identify the key factors influencing booking cancellations.
- Explore customer demographics and booking trends.
- Evaluate revenue-related metrics and seasonal patterns.
- Generate actionable business recommendations to improve hotel performance.
